Famille de taxe
===============

.. toctree::
	:maxdepth: 1
	:titlesonly:

Cette option vous permet de gérer les familles de taxe.
Les taxes sont utilisées au niveau du module export lors de la génération de la formule provisoire

.. image:: ../_img/referentiels/familletaxe1.png
	:alt: liste
	:align: center
	:height: 224
	:width: 800
	
   
**Edition de la fiche : Famille des tases**

Toutes les zones de cet écran sont obligatoire.

	* **Code** : indiquez le code de la famille. Ce code doit être unique.
	* **Libelle** : indiquez la désignation de la famille.
